# 这是一个能自动生成android studio ShapeDrawable文件模板插件

   - **插件使用方法**
      1. 选中一个文件夹
      2. 右键单击，选择"Generate ShapeDrawable Template" 菜单项
      3. 在选中的文件目录下生成ShapeDrawable成功。
